# okamu dotfiles

https://github.com/okamuroshogo/dotfiles-light

## installaton
### Dotfiles
```
# Install homesick
gem install homesick

# Clone repository
homesick clone okamuroshogo/dotfiles

# Link dotfiles
homesick symlink dotfiles
```

### Install Homebrew
```
cd ~/.homesick/repos/dotfiles/
./.brew_install
```

### Install anyenv environment
```
cd ~/.homesick/repos/dotfiles/
./.anyenv_install
```

### Setup osx
```
cd ~/.homesick/repos/dotfiles/
./.osx
```

### Install latest env
```
cd ~/.homesick/repos/dotfiles/
./.latest_env_install
```
